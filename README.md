# MERN Stack Web Development - Assignment 3

## About the Project

This is my third assignment in the MERN Stack Web Development course.

The purpose of this project is to practice basic HTML and CSS concepts, including:

- Heading tags (`<h1>` to `<h6>`)
- Paragraph tag (`<p>`)
- Ordered and unordered lists
- Anchor tags (`<a>`)
- Inline CSS
- External CSS

This is a simple dummy website created for learning and practice purposes.

- CSS3

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Rohit98Data55Analyst/Assignment3.git
```

### 2. Open the project folder

```bash
cd Assignment3
```

### 3. Run the project

Open `index.html` in your preferred web browser.

**OR**

Open the project in **Visual Studio Code** and use the **Live Server** extension.
